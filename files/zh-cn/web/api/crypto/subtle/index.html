Crypto.subtle 是一个只读属性，返回一个 {{domxref("SubtleCrypto")}} 对象允许做一个些加密操作。

语法

var crypto = crypto.subtle;

值

一个可用于与Web Crypto API的低级加密功能交互的 {{domxref("SubtleCrypto")}} 对象。
